Gene to Literature Integrative Database (GLID)


Imagine you are interested in a gene or protein. People call genes and proteins in different ways, which is a nightmare for anyone who is interested in studying its function, evolution or clinical implications. In addition, phylogenetically related genes (potentially interesting if you want to study the function or evolution of the gene across organisms) do not always share nomenclature.

At the end you may query PubMed in a partial way, missing information you would like to have. GLID aims to solve this nightmare.

We have mined nomenclature data of many sources (HGNC, UniProt, NCBI, OrthoDB and others, using this code) and built an integrative database with this information. We have provided a user-friendly interface to navigate across dfferent gene/protein names, select them and query pubmed. The database offers the possibility to add keywords to the search and displaying the literature matching your patterns.

Futhermore, we have reasoned that anyone interested in a gene/protein may also be interested in those genes with a similar function (for example in the same species). We have provided two sections (similar proteins and similar function) that provide an approach to this information.

  • On the one hand, two proteins that share the exact same domains are likely to have the same function, and the section "similar proteins" provides this information from the Pfam database.

  • On the other hand, two genes that share a given number of Gene Ontology (GO) terms are also likely to have similar function . We have provided these relationships for each of the functional terms (Molecular Function, Biological Process and Cellular Component) in the "similar function" section. Note that this approach is limited to the fact that well-studied genes have more GO-terms and that some GO-terms are very close (in the GO relationship) but they would be considered different in our analysis.


NOTE: This is a pilot database, which means that only a reduced number of species (see below) has been included. This has an important impact in the database content of the phylogenetically-related genes, obtained from OrthoDB. Many genes won't have any such partners because we have not included their corresponding phylogenetically related species. Keep this in mind when doing your search.

How to query GLIDdb


  Enter your query

Different types of query are supported by GLID. In the query form, the following terms can be introduced:

  • a gene name (short or long, ether being the reccomended or an alternative name)
  • a protein name (short or long or an alternative name)
  • an EntrezGene ID or a UniProt ID
 The search is case-insensitive, meaning that upper and lower case will be ignored and treated as equal.

Here there are some examples of usage: CDK2, cdk2, Cyclin-dependent kinase 2, 1017, P24041.


Search options


  Look for the recommended gene/protein name

Enter your query gene/protein name or ID, and obtain the recommended gene/protein name, if there is one. The list of recommended names are retrieved from the UniProt consortium and HGNC (HUGO Gene Nomenclature Commitee).

From that list, you can select as many as you wish and then proceed to download a JSON file with them and/or query PubMed to maximise your literature search results.


  Find all the synonyms of your gene/protein

Enter your query gene/protein name or ID, and obtain a list of all the gene/protein synonyms. The list of all synonyms is retrieved from the UniProt consortium.

From that list, you can select as many as you wish and then proceed to download a JSON file with them and/or query PubMed to maximise your literature search results.


  Find phylogenetically related genes from OrthoDB

Enter your query gene/protein name or ID and a specie to retrieve a list of all phylogenetically related genes annotated inside an OrthoDB group. For a better clarity, we provide a list of genes that are phylogenetically related to the query, as well as which are the species that include each of these genes. Go to the OrthoDB help page for more information about how OrthoDB clusters phylogenetically related genes.

As a consequence of selecting a reduced number of species for this pilot version, only approximately a 10% of the genes are showing philogenetically related genes.

From that list, you can select as many as you wish and then proceed to download a JSON file with them and/or query PubMed to maximise your literature search results.


  Find similar proteins according to PFAM domains

Enter your query gene/protein name or ID, and retrieve a list of all the described proteins which have the exact same PFAM domains.

From that list, you can select as many as you wish and then proceed to download a JSON file with them and/or query PubMed to maximise your literature search results.


  Find proteins with similar function according Gene Ontology (GO)

Enter your query gene/protein name or ID, and retrieve a list of all the described proteins which, for each of the different types of Gene Ontology terms (Cellular Component, Biological Process, Molecular Function), have at least a 80% of intersection with the query.

From that list, you can select as many as you wish and then proceed to download a JSON file with them and/or query PubMed to maximise your literature search results.


Restrict to one specie name or taxonomy group


GLID allows to select either a single specie or a taxonomic group from two predefined sets: the specie names list and the taxonomic groups list. Regarding the specie names list, 14 species were choosen for this pilot web application. Regarding the taxonomic groups list, the taxonomic hierarchical classification division defined by the NCBI was used.

 In the corresponding forms, start typing to see if your specie or taxonomic group of interest is supported.

Query to PubMed


Once your results have been reported considering your previously selected search options, you can choose as many as these gene/protein names (short and long terms) to be submitted on PubMed. Additionally, before submitting them, you can add some extra keywords if necessary.


Download/Upload your results


JSON file download
Once your results have been reported considering your previously selected search options, you can select as many as you wish and then proceed to download a JSON file.

JSON file upload
At the home page, if you have previously performed a search and download its associated JSON array file, you can upload this file to directly submit its content to PubMed.
