How to query GLID database

Enter a name

Different types of query are supported by GLID. You can enter a gene name (short or long), a protein name (short or long), an EntrezGene ID or a UniProt ID in the query form. The search is case-insensitive, meaning that upper and lower case will be ignored and treated as equal. Here there are some examples of usage: CDK2, cdk2, Cyclin-dependent kinase 2, 1017, P24041.


Choose what to search

GLID can perform different types of search depending on your interests.


Look for the recommended name of a gene/protein

Just enter your query, click the option Recommended gene name and select the species. You will obtain the recommended name, if there is one. Recommended names are retrieved from the UniProt consortium and HGNC (HUGO Gene Nomenclature Commitee).


Find all the synonyms of your gene/protein

Enter a query gene/protein name or ID and obtain a list of all the synonyms. From that list, you can select as many as you wish and then proceed to download a JSON file with them and/or query PubMed to maximise your literature search results. The list of all synonyms is retrieved from the UniProt consortium.


Find phylogenetically related genes from OrthoDB

Enter a query gene/protein name and a species to retrieve a list of all phylogenetically related genes annotated inside an OrthoDB group. From that list, you can select as many as you wish and then proceed to download a file with them and/or query PubMed to maximise your literature search results.


Find similar proteins according to PFAM domains

Enter a query gene/protein name or ID and retrieve a list of all the described proteins which have the exact same PFAM domains. From that list, you can select as many as you wish and then proceed to download a file with them and/or query PubMed to maximise your literature search results.


Find proteins with similar function according Gene Ontology

Enter a query gene/protein name or ID and retrieve a list of all the described proteins which, for all the different types of Gene Ontology terms, have at least a 80% of intersection with the query. From that list, you can select as many as you wish and then proceed to download a file with them and/or query PubMed to maximise your literature search results.


Restrict to one/many species name or taxonomy

GLID allows to select a single specie from a list or either a taxonomic group. Start typing to see if your specie or taxonomic group of interest is supported.

Here we provide the complete list:

Species: Mus musculus, Saccharomyces cerevisiae, Schizosaccharomyces pombe, Pan troglodytes, Bacillus subtilis, Gallus gallus, Xenopus laevis, Nicotiana tabacum, Escherichia coli, Caenorhabditis elegans, Drosophila melanogaster, Danio rerio, Homo sapiens, Bos taurus.

Taxonomic groups: Bacteria, Invertebrates, Mammals, Plants and Fungi, Primates, Rodents, Vertebrates.
