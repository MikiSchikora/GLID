How to query GLID database

Enter a name

You can enter a gene/protein name or Ensembl/UniProt ID or any string of characters in the query form. (better say "the user"????) The search is case-insensitive (????), meaning that upper and lower case will be ignored. (wildcards for substrings??????)

Choose what to search

You can perform different types of search depending on your interests.

Look for the recommended name of a gene/protein

Just enter the name of the gene/protein, click the option "Recommended gene name" and select the species. You will obtain the recommended name, if there is one (???).

Find all the synonyms of your gene/protein

Enter a gene/protein name or ID and obtain a list of all the synonyms. From that list, you can select as many as you wish and then proceed to download a file with them and/or query PubMed to maximise your literature search results.

Find orthologues from OrthoDB

Enter a gene/protein name and a species to retrieve a list of all orthologous genes annotated in OrthoDB. From that list, you can select as many as you wish and then proceed to download a file with them and/or query PubMed to maximise your literature search results.

Find similar proteins according to PFAM domains

Enter a gene/protein name or ID and retrieve a list of all the described proteins which have the same PFAM domains. From that list, you can select as many as you wish and then proceed to download a file with them and/or query PubMed to maximise your literature search results.
